justify-content:

Adding the justify-content property to the parent container positions child items along the main axis.

When in the default row orientation, the child items will be horizontal, from left to right. ➡︎.

If the orientation is changed to column, the child items will be vertical, from top to bottom ⬇︎.

The default position is flex-start.
