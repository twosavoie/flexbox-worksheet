Flexbox Rules

  • To use Flexbox, add display: flex; to the parent container.
  • Changing the display property to flex gives us access to Flexbox properties
  • Flexbox has both container and item properties
  • Flexbox works with width and height properties. The Flexbox properties say where the content is positioned and how to divvy up any space remaining in the container.
  • Adding display: flex; moves all of the flex child items to a row. Each item is only as wide as its content. Adding flex-flow: row wrap; or flex-wrap: wrap; along with a width for the content creates a flexible, responsive layout.
  • Flexbox is both horizontally and vertically orientated, allowing for complex layouts.

How to complete the worksheet

  • In the main.css file, use the comments to find the CSS for each section.
  • Start by centering the content-wrapper for the section by adding display: flex;. Then center the content-wrapper with justify-content: center;.
    • The "content-wrapper" has a purple background Purple.
  • For each Flexbox property's value, add display: flex; and the Flexbox property and value in the class name.
    • For instance in the .justify-content-center-parent-container code block, add display: flex; and justify-content: center;.
    • The content boxes are blue Blue.

justify-content:

Adding the justify-content property to the parent container positions child flex items along the main axis.

When in the default row orientation, the child flex items will be horizontal, from left to right ➡︎.

If the orientation is changed to column, the child flex items will be vertical, from top to bottom ⬇︎.

The default position is flex-start.

center

Positions elements in the center of the parent container. Any space within the container, not taken up by the content, will be evenly divided on the outside of the items without any spacing between the elements.

space-between

The first and last item touch the edge of the parent container. Any space within the container, not taken up by the content will be divided evenly between the items.

space-around

The items are positioned within the container with the extra space divided evenly around each item.

space-evenly

This value is very similar to space-around. The extra space is divided so that there is the same amount of space between the edge of the parent container and each item.

flex-start

Positions elements at the beginning of the container. If our main axis is horizontal, it will position it to the left. If it is vertical, it will position it at the top.

flex-end

Positions elements at the end of the container. If our main axis is horizontal, it will position it to the right. If our main axis is vertical, it will position it at the bottom.

align-items:

Adding the align-items property to the parent container positions child items along the cross axis.

When in the default row orientation, the cross axis will be vertical, from top to bottom ⬇︎.

If the orientation is changed to column, the cross axis will be horizontal, from left to right ➡︎.

The default value is stretch.

flex:

The flex property is three properties in one. It include flex-grow, flex-shrink, and flex-basis

The flex property is applied to elements along the main axis.

When display: flex; is added to a parent container, the content's size and the rate it shrinks or grows when the browser window changes size is set.

The default values are flex: 0 1 auto;.

default

The default flex values of flex: 0 1 auto; mean that the flex-grow property has a value of 0, the flex-shrink property has a value of 1, and the flex-basis property as a value of auto.

This means that the content is the size its set to in CSS (the flex-basis. Each child flex item will shrink at the same rate flex-shrink. And the child flex items will not grow beyond the width they've been set to flex-grow.

In this example, the "content-wrapper is 90% of the width of the browser window to a max-width of 1200px. Let's look at the desktop version to make the math a bit easier.

flex-basis

The flex-basis property sets a width to a flex child item. flex-basis is very similar to the width property. If a flex child item has a width set to it with flex-basis, as long as the container has that width available, the child flex element will be that width. flex-basis also overwrites the width property.

flex-basis: 800px;

flex-shrink

The flex-shrink property controls how a flex child item shrinks when the browser window becomes more narrow. If the flex-shrink property's value is 0, the flex child item will not shrink at all. If the flex-shrink is set to 2, and the other flex child items have a flex-shrink value of 1, the flex child item will shrink twice as much.

The width of the second box is mobile - 200px, tablet - 400px, and desktop - 800px.

flex-shrink: 0;

flex-grow

The flex-grow property controls how much of the extra space within a parent flex container a flex child item takes up. The flex-grow values of all of the flex child items are added together and the extra space is divided into shares. If a single flex child item has a value of 1 and all others have a value of 0, the flex child item with a value of 1 takes all of the extra space. If there are three flex child items and two of them have a flex-grow value of 1, the extra space is divided into two shares and each gets a single share of the extra space.

flex-grow: 1;

flex-basis column

flex-basis sets the box's height as long as there's enough height.

When the height of the parent container is less than the flex-basis value, the child flex item will shrink at the rate set by the flex-shrink value.

flex-basis: 800px;

flex-shrink column

The height of the second box is mobile and tablet - 75px, and desktop - 120px.

If the height of the parent container shrinks, the second column's height will not change and will overflow its container. Try it out in DevTools.

flex-shrink: 0;

flex-grow column

The height of the second box is set with the flex-grow property, taking up any extra space in the parent flex container.

If more than one child flex item has a flex-grow value greater than 0, it will share the extra space.

flex-grow: 1;
